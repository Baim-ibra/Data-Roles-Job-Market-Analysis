# 📊 Data Analyst Job Market Analysis: An End-to-End SQL Project

## 🎯 Project Overview
This project performs an end-to-end analysis of the Data Analyst job market. By utilizing SQL Server for data processing and analysis, I transformed raw job posting data into actionable business insights. The goal is to identify trends in salary, identify high-demand skills, and determine the "sweet spot" for professional development.

---

## 🛠️ Project Workflow
1. **Creating Database:** Setting up the environment to house structured job data.
2. **Generating Tables:** Designing schemas to manage job postings, companies, and skill sets.
3. **Importing Data:** Loading raw datasets into the SQL Server database.
4. **Generating Queries:** Writing complex SQL queries (CTEs, JOINs, Aggregations) to extract insights.
5. **Business Decision Support:** Creating data-driven narratives for career growth.
